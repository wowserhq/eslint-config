## Usage

To install `@wowserhq/eslint-config` and its dependencies:

```shell
npm install @wowserhq/eslint-config --save-dev
npm install eslint-plugin-import --save-dev
npm install eslint --save-dev
```

Tweak `.eslintrc`:

```json
{
  "extends": "@wowserhq/eslint-config"
}
```
